# Jeden ukryty Winny wśród Podejrzanych

Każda runda ma dokładnie jednego Winnego, który otrzymuje niepełne albo zniekształcone Alibi; pozostali Podejrzani są Niewinni i znają pełne Alibi, ale nie znają wzajemnych ról. Wybieramy model „kto kłamie” zamiast jednej współpracującej drużyny Podejrzanych, aby Detektyw miał obiektywny cel i ponosił ryzyko wskazania niewłaściwej osoby.
