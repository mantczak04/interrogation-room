# Sprawy powstają z ręcznie przygotowanych modułów

Przestępstwa i Alibi są składane podczas Rundy z kompatybilnych, ręcznie napisanych modułów takich jak miejsce, czas, transport, napotkana osoba, przedmiot i dziwne zdarzenie. Rezygnujemy z generowania treści przez AI podczas gry, aby zachować kontrolę nad spójnością, humorem, poziomem trudności i lokalizacją oraz uniezależnić podstawową rozgrywkę od zewnętrznej usługi.
