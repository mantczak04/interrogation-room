# Jedna Egzekucja kończy rundę

Detektyw ma w rundzie dokładnie jedną Egzekucję, która natychmiast ją kończy: wyeliminowanie Winnego daje Detektywowi zwycięstwo, a wyeliminowanie Niewinnego oznacza jego przegraną. Brak możliwości poprawienia strzału zachowuje ryzyko decyzji i pozwala jednoznacznie rozliczyć Przetrwanie oraz Sekretne Cele wszystkich Podejrzanych.
